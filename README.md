# dotfiles

Personal Arch Linux rice — Hyprland + HyDE, built for an AI/cybersecurity workflow.

## Stack

| Layer | Tool |
|---|---|
| Compositor | Hyprland (managed by HyDE) |
| Bar | Waybar |
| Terminal | Kitty |
| Shell | Zsh + Oh My Zsh + Starship |
| Multiplexer | Tmux + TPM |
| Editor | Neovim / LazyVim |
| Notifications | Dunst |
| Launcher | Rofi |
| File manager | Dolphin / Yazi |
| System monitor | Btop |
| Fetch | Fastfetch |

## Install

### Prerequisites

1. Arch Linux
2. HyDE installed → https://github.com/HyDE-Project/HyDE
3. Packages: `zsh oh-my-zsh-git starship tmux kitty btop fastfetch eza zoxide fzf bat fd ripgrep lazygit lazydocker`

### Clone and run

```bash
git clone https://github.com/Imad-Oute/dotfiles ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

The script symlinks everything into place and backs up any existing files to `~/.dotfiles-backup-<timestamp>`.

### Wallpapers

Wallpapers are not in the repo. See [wallpapers.md](wallpapers.md) for Wallhaven IDs and re-download instructions.

### Tmux plugins

After install, open tmux and press `prefix + I` (that's `Ctrl+Space + I`) to install TPM plugins.

## What's tracked

```
dotfiles/
├── .config/
│   ├── hypr/              Hyprland — userprefs, keybindings, windowrules, animations...
│   ├── hyde/
│   │   ├── config.toml    HyDE behavior
│   │   └── themes/        Claude Dark · Neural · Phantom · Entropy
│   ├── waybar/            user-style.css overrides
│   ├── kitty/             kitty.conf base config
│   ├── nvim/              Neovim / LazyVim config
│   ├── starship/          starship.toml prompt
│   ├── btop/              btop.conf
│   ├── fastfetch/         fastfetch config + logo
│   └── dunst/             notification config
└── home/
    ├── .zshrc             shell aliases, PATH, tool init
    └── .tmux.conf         tmux + TPM config
```

## What's NOT tracked

- `~/.config/hypr/themes/` — auto-generated by HyDE on every theme/wallpaper switch
- `~/.config/waybar/config.jsonc`, `style.css` — auto-generated by wbarconfgen
- `~/.config/kitty/theme.conf` — auto-generated by HyDE theme switcher
- `~/.zshrc.local` — machine-local secrets (API keys etc.)
- Wallpaper images — see [wallpapers.md](wallpapers.md)
