# Tridactyl Keybinds

## Scrolling
| Key | Action |
|-----|--------|
| `j / k` | scroll down / up (5 lines) |
| `d / u` | scroll half page down / up |
| `gg / G` | top / bottom of page |

## Navigation
| Key | Action |
|-----|--------|
| `H / L` | back / forward |
| `r / R` | reload / hard reload |

## Tabs
| Key | Action |
|-----|--------|
| `J / K` | next / prev tab |
| `x` | close tab |
| `u` | undo close tab |
| `t` | open new tab (with URL bar) |
| `T` | open current URL in new tab |
| `b` | fuzzy search open tabs |

## Opening URLs
| Key | Action |
|-----|--------|
| `o / s` | open URL (same tab) |
| `O / S` | open URL (new tab) |

## Hints (click without mouse)
| Key | Action |
|-----|--------|
| `f` | hint links → open same tab |
| `F` | hint links → open new tab |
| `gi` | focus first input on page |

## Yank / Clipboard
| Key | Action |
|-----|--------|
| `yy` | yank current URL |
| `yt` | yank page title |
| `p / P` | open clipboard URL (same / new tab) |

## Tree Style Tab — tree structure
| Key | Action |
|-----|--------|
| `>` | expand current tab's tree |
| `<` | collapse current tab's tree |
| `]]` | indent — make child of tab above |
| `[[` | outdent — promote to parent level |

## Tree Style Tab — tree navigation
| Key | Action |
|-----|--------|
| `Alt+j` | next sibling (skip children) |
| `Alt+k` | prev sibling (skip children) |
| `Alt+p` | jump to parent tab |
| `Alt+c` | jump to first child tab |
| `Alt+z` | collapse all trees except current |
| `Alt+b` | toggle TST sidebar |

## Misc
| Key | Action |
|-----|--------|
| `/ n N` | find / next / prev |
| `zi / zo / z=` | zoom in / out / reset |
| `Alt+r` | reader mode |
| `Alt+m` | mute/unmute tab |
| `Alt+s` | reload tridactylrc |
| `Space` | passthrough next key to page |

## Insert mode
| Key | Action |
|-----|--------|
| `jk` | escape insert mode → normal mode |

## Quickmarks
| Key | Action |
|-----|--------|
| `go<key>` | open quickmark (same tab) |
| `gn<key>` | open quickmark (new tab) |
| Add: `:quickmark g https://github.com` | |
